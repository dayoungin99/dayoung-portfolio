Backend Data Pipeline

An end-to-end ETL pipeline that ingests raw data, applies transformations into a clean schema, and loads the result into a query-ready database.

Technologies used

  • Python
  • PostgreSQL
  • Docker
  • Pandas

Features

  • Schema-first transformations with clear input/output contracts
  • Idempotent loading to safely re-run daily jobs
  • Structured logging for easier debugging and observability
